## Immo Investment Kalkulation

Ein kleines React + TypeScript Tool zur Kalkulation einer Immobilieninvestition, inspiriert vom Layout des 1:1 Immoguide Kalkulationstools (`https://calctool.1zu1-immoguide.de/`).

### Entwicklung starten

```bash
npm install
npm run dev
```

Dann im Browser `http://localhost:5173` öffnen.

### Technologie-Stack

- React 18 + TypeScript
- Vite als Dev-Server und Bundler
- Moderne, responsive UI mit CSS Grid/Flexbox
